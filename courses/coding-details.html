Description

Welcome to Code Explorers, where young minds embark on a thrilling journey into the world of programming! This dynamic and engaging coding course is specially designed for Children aged 8 to 12, fostering creativity, problem-solving skills, and a love for technology

Course Overview

Embark on a thrilling coding adventure with Code Explorers! Over 24 weeks, young minds aged 6 - 16 will journey through six exciting modules, from unraveling the mysteries of programming basics and crafting digital masterpieces to building interactive games, websites, and solving real-world challenges.

What Are Pick Up And Drop Off Times For My Child?

Students may arrive 15 minutes prior to the scheduled start time and they must be picked up no later than 15 minutes after the end time. Unfortunately, we do not offer extended care beyond these times for our Children woodworking classes.

Full (100%) refund if canceled at least 21 days in advance of the start date, half (50%) refund for cancellations 14 days in advance of the start date, and no refunds less than 14 days in advance of the start date. Summer camp: Full (100%) refund if canceled.

Unveiling the Coding Universe

  1. Introduction to the world of coding and computational thinking.
  2. Understanding algorithms and breaking down problems into smaller steps.
  3. Hands-on activities using visual programming languages to create simple programs and games.

Craftting Digital Masterpiace

  1. Exploring the artistic side of coding through projects like drawing and animation.
  2. Introduction to storytelling through code, where Children bring their narratives to life.
  3. Group projects to encourage collaboration and creative expression.

Building Fun and Excitement

  1. Basic game design principles and coding interactive games.
  2. Exploring game mechanics and player engagement.
  3. Team-based projects to create unique and exciting games.

Building Interactive Website and Pages

  1. Introduction to HTML and CSS for web development.
  2. Creating simple web pages and adding interactivity using JavaScript.
  3. Developing a basic understanding of web design and user experience.

Cracking the code challenge

  1. Developing critical thinking and problem-solving skills through coding challenges and puzzles.
  2. Collaborative problem-solving projects to apply learned concepts in real-world scenarios.
  3. Introduction to debugging and error resolution.

Exploring Advanced Concepts

  1. Building on foundational programming concepts (conditional statements, loops, functions).
  2. Final project development, where Children showcase their skills and creativity.
  3. Graduation ceremony with project presentations to parents and peers.

Course Highlight

  1. Small class sizes to ensure personalized attention and support.
  2. Graduation ceremony where Children showcase their projects to parents and peers.
  3. Emphasis on creativity and exploration, encouraging Children to express themselves through code.
No prior coding experience is required. Enthusiasm, curiosity, and a passion for learning are the only prerequisites!
